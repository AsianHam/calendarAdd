The purpose of this program is to give or take away (edit) access to any Shared Google Calendar.

** credentials.json can be generated from the link below:
https://developers.google.com/calendar/quickstart/python

** this program was intended to be created as an exe file using 'auto-py-to-exe'

For full documentation regarding the Google Calendar API, see the link below:
https://developers.google.com/calendar


------------------------------------------------ How To Use This Program -------------------------------------------------

- In the main.py file, edit the 'newDict' dictionary with the name and calendar ID of the calendars you want on the GUI.

- Run the main file to start the GUI

- If running for the first time, the program will open a new page in your web browser to generate a token.pickle file

- Make sure both the token and the credentials file are in the same directory as the rest of the files

- Enter the email addresses of the people whose access you want to change, separated by commas or a space

- Select the calendars you want to change access to

- Press submit

- The list button can be pressed to see who has access to which calendars
